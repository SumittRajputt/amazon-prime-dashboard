# Amazon Prime Content Analysis Dashboard

## 📊 Project Overview
This project presents an interactive Power BI dashboard analyzing Amazon Prime content including movies and TV shows.

## 🔍 Key Insights
- Movies dominate the platform (~85%)
- Drama and Comedy are the most common genres
- Content production increased significantly after 2000
- Ratings mostly fall between 6–8
- Few actors appear frequently across titles

## 🛠 Tools Used
- Power BI
- Data Cleaning & Visualization
